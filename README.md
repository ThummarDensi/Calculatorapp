
##  CALCULATOR ANDROID APP

Calculator Android App that does the following operations:
  
- Addition
  
- Subtraction
  
- Multiplication
  
- Division

## APPLICATION DEVELOPMENT
 
- Programming Information:
   
  - JAVA
   
  - Android Studio
 
- Fonts:
   
  - [Orbitron](https://fonts.google.com/specimen/Orbitron)
